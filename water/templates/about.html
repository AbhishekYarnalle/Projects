💧 About Project

Drinking Water Potability Classification using Machine Learning is a smart web-based AI system developed to determine whether water is safe for drinking based on multiple chemical and physical parameters.


The system leverages a Random Forest Machine Learning model trained on the water potability dataset to perform accurate classification. It integrates prediction, analytics, visualization, and secure user tracking within a unified full-stack platform.


This project contributes to smart environmental monitoring and public health awareness while demonstrating real-world deployment of AI systems.


• AI-based water safety prediction
• Secure login and prediction history tracking
• Dataset visualization and analytics dashboards
• Automated ML model training and deployment
• Database + Excel logging system
• Interactive futuristic user interface
