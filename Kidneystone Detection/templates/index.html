System Features

AI Prediction

This system analyzes kidney X-ray images using artificial intelligence models to detect kidney stones quickly and accurately.

Medical Analysis

The system provides graphical analysis and prediction statistics to help understand kidney stone detection results.

Health Solutions

Users receive lifestyle and diet recommendations that help reduce the risk of kidney stone formation.

Doctor Recommendations

If kidney stones are detected, the system suggests consulting medical specialists for proper diagnosis and treatment.

How The System Works

Step 1

Upload a kidney X-ray image to the system.

Step 2

The machine learning model processes the image.

Step 3

The system detects whether kidney stones exist.

Step 4

Results and recommendations are displayed.
