Kidney Stone Detection System

This project is an AI-powered medical imaging system that detects kidney stones from X-ray images using machine learning algorithms. The goal of the system is to assist doctors and patients in early diagnosis of kidney stones.

Project Objectives

  • Detect kidney stones using AI image analysis
  • Assist medical professionals with faster diagnosis
  • Provide health recommendations for patients
  • Visualize prediction results and statistics

Technologies Used

  • Python
  • Flask Web Framework
  • Machine Learning (Random Forest / Deep Learning)
  • OpenCV & Image Processing
  • SQLite Database
  • HTML, CSS, Bootstrap

How the System Works

Users upload kidney X-ray images through the prediction page. The trained machine learning model analyzes the image and predicts whether a kidney stone is present or not. The system then displays prediction results along with recommendations.
